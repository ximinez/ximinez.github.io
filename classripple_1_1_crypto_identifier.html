Detailed Description

template<std::size_t Size, std::uint8_t Token, bool Checked>
class ripple::CryptoIdentifier< Size, Token, Checked >

Shared IdentifierType traits for Ripple crypto identifiers.

Template Parameters
SizeThe number of bytes in the identifier, exclusive of version, checksum, or padding.
TokenA byte prepended to the binary data that distinguishes the type of identifier.
CheckedA bool indicating whether or not the string representation includes an appended a four byte checksum on the data including the Token.
