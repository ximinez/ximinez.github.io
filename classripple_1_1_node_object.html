Detailed Description

A simple object that the Ledger uses to store entries.

NodeObjects are comprised of a type, a hash, a ledger index and a blob. They can be uniquely identified by the hash, which is a SHA 256 of the blob. The blob is a variable length block of serialized data. The type identifies what the blob contains.

Note
No checking is performed to make sure the hash matches the data.
