forceValidity

Sets the validity of a given transaction in the cache.

Synopsis 
void
forceValidity(
    HashRouter& router,
    uint256 const& txid,
    Validity validity);
Description

Use with extreme care.

Remarks

Can only raise the validity to a more valid state, and can not override anything cached bad.
