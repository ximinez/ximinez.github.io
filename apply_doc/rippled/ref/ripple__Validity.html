Validity

Describes the pre-processing validity of a transaction.

Synopsis 
enum Validity

Values

Name

Description

SigBad

Signature is bad. Didn't do local checks.

SigGoodOnly

Signature is good, but local checks fail.

Valid

Signature and local checks are good / passed.
