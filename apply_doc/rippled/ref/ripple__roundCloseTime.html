roundCloseTime

Calculates the close time for a ledger, given a close time resolution.

Synopsis 
template<
    class time_point>
time_point
roundCloseTime(
    time_point closeTime,
    typename time_point::duration closeResolution);
Description
Parameters

Name

Description

closeTime

The time to be rouned.

closeResolution

The resolution

Return Value

closeTime rounded to the nearest multiple of closeResolution. Rounds up if closeTime is midway between multiples of closeResolution.

Header

#include <ripple/consensus/LedgerTiming.h>
