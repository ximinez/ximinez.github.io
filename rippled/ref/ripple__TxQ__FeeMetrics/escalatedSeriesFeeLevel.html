TxQ::FeeMetrics::escalatedSeriesFeeLevel

Returns the total fee level for all transactions in a series.

Synopsis 
std::pair< bool, std::uint64_t >
escalatedSeriesFeeLevel(
    OpenView const& view,
    std::size_t extraCount,
    std::size_t seriesSize) const;
Description

Assumes that there are already more than txnsExpected_ txns in the view. If there aren't, the math still works, but the level will be high.

Returns: A std::pair as returned from mulDiv indicating whether the calculation result is safe.
