TxQ::apply

Add a new transaction to the open ledger, hold it in the queue, or reject it.

Synopsis 
std::pair< TER, bool >
apply(
    Application& app,
    OpenView& view,
    std::shared_ptr< STTx const > const& tx,
    ApplyFlags flags,
    beast::Journal j);
Description
Return Value

A pair with the TER and a bool indicating whether or not the transaction was applied. If the transaction is queued, will return { terQUEUED, false }.
