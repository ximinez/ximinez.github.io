Detailed Description

template<typename c_Key, typename c_Data, typename c_Hash = beast::uhash<>>
class ripple::SyncUnorderedMapType< c_Key, c_Data, c_Hash >

This is a synchronized unordered map.

It is useful for cases where an unordered map contains all or a subset of an unchanging data set.
