Descend along the specified branch On invocation, nodeID must be the ID of this node Returns false if there is no node down that branch Otherwise, returns true and fills in the node's ID and hash.

Parameters
branchThe branch to descend [0, 15]
nodeIDOn entry the ID of the parent. On exit the ID of the child
nodeHashOn exit the hash of the child node.
Returns
true if nodeID and nodeHash are altered.
