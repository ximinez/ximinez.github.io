Detailed Description

The SSL contexts used by Ripple.

This is what Ripple uses for its secure connections. The ECDSA curve parameters are predefined and verified to be secure. The context is set to sslv23, Transport Layer Security / General. This is primarily used for peer to peer servers that don't care about certificates or identity verification.

std::string ripple::RippleSSLContext::getRawDHParams ( int  keySize)
static

Retrieve raw DH parameters.

This is in the format expected by the OpenSSL function d2i_DHparams. The vector is binary. An empty vector means the key size is unsupported.

Note
The string may contain nulls in the middle. Use size() to determine the actual size.
